# bikeshare.py

bikeshare.py is a Python script that provides descriptive statistics about bike share use in Chicago, New York City, and Washington.

## Purpose and Data 

This script is a project in Progrogramming for Data Analysis cource at [Udacity](https://www.udacity.com/). 

Original data was provided by [Motivate](https://www.motivateco.com/). Raw data were modified for the purpose of the course. 


Data date range: January 2017 - June 2017 

Following files from Udacity course is required to run the script: chicago.csv, new_york_city.csv, washington.csv
